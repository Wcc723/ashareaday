2013-winter

假日來點簡單的~，canvas是一種點陣繪圖技術，所以來複製或是繪製一張影像檔不會有什麼問題。

繪製外部Jpg

首先當然要有一張圖，然後指定一個路徑去擷取他。

  var _canvas = document.getElementById('canvas');
  
  if (_canvas.getContext) {
    var ctx = _canvas.getContext('2d');
    draw();
  }

  function draw(){
  	var img = new Image(); //建立一個Image物件
  	img.src = 'images/2013-winter/cd2.jpg';  //圖片路徑
  	img.onload = function(){  //img物件載入  		
  		ctx.canvas.width = img.width;  //取得來源的寬
  		ctx.canvas.height = img.height;  //取得來源的高
  		ctx.drawImage(img,0,0);  //繪製
  	};
 	
  }
