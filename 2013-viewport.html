番外篇 Viewport

viewport是屬於html的meta屬性，它的功能是決定網頁在裝置上所呈現的大小，可以調整行動裝置的顯示能夠符合各種大小。

<meta name="viewport" content="initial-scale=1.0, width=device-width" />

meta是放在html頁面的程式碼head的內部，name="viewport"後開始設定各項的屬性值，而相關屬性如下。

參考：http://devdocs.io/html/meta
Value Possible values Description
width 一個整數值或是設定device-width(裝置寬度) 定義網頁所呈現的寬度值
height 一個整數值或是設定 device-height(裝置高度) 定義網頁所呈現的高度值
initial-scale 一個數值從0.0 到 10.0 之間 定義縮放百分比
maximum-scale 一個數值從0.0 到 10.0 之間 定義縮放百分比 最大值
minimum-scale 一個數值從0.0 到 10.0 之間 定義縮放百分比 最小值
user-scalable a boolean value (yes or no) If set to no, the user is not able to zoom or in the webpage. Default value is yes.
