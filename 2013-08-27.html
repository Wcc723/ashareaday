Coffee Script 反向編譯 無痛學習

來源 : http://js2coffee.org/

Scope.prototype.find = function(name, options) {
    if (this.check(name, options)) {
        return true;
    }
    
    this.add(name, 'var');
    return false;
}
//這是一打開的預設js

# Type here! 
Scope::find = (name, options) ->
  return true  if @check(name, options)
  @add name, "var"
  false

#這是編譯成coffee後的

再透過這樣的編譯，我們可以簡單理解一些事情，並且嘗試學習coffee script。

  1. 巢狀結構代替大括號{}
  2. function 在 coffee寫作 ->
  3. 註解 在 coffee寫作 #
  4. this. 在 coffee寫作 @
  5. 小括號 在coffee不一定要打，只要空格隔開就可以了
  6. 可以先不定義變數(建議還是先定義)
